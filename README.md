Shopping Data Analytics Project
Project Overview

This project focuses on analyzing shopping data to uncover meaningful business insights. 
The raw dataset was cleaned and preprocessed using Python and the Pandas library to handle missing values, 
remove inconsistencies, and improve data quality.
Tools Used:
*Python
*Pandas
*Excel
*Power BI
Project Workflow:
Imported the raw shopping dataset into Python.
Performed data cleaning and preprocessing using Pandas.
Exported the cleaned dataset to Excel.
Connected the dataset to Power BI.
Created interactive dashboards and visualizations.
Key Insights:
Sales trends and performance analysis
Customer purchasing behavior
Product performance evaluation
Revenue and transaction analysis
Outcome

This project demonstrates the complete data analytics workflow, from data cleaning and preparation to visualization and insight generation, using Python and Power BI.
